## Generación de documentación con IA
Se ha añadido una herramienta para generar documentación Markdown de objetos PL/pgSQL.

- Script: `scripts/generate_plpgsql_documentation.py`
- Salida: `docs/plpgsql_documentation.md`
- Opción IA: `--ai` con `OPENAI_API_KEY`
